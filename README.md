# 28. Oktober
During this session, we explored several sensors, such as a tilt switch, a potentiometer, and a photoresistor. We also worked with Arduino functions like analogRead, analogWrite, and used the Serial Monitor to observe values from the sensors. One activity involved using a potentiometer to adjust how bright an LED lights up. Understanding the analog values was a bit confusing for me at first, but with some practice it started to become clearer.

# 4. November
In this session, we worked with several new components. For example, we learned about different types of buzzers, including active and passive ones, and how they can be used to create sounds with the Arduino. We also experimented with an ultrasonic distance sensor to measure how far away objects are. During this activity, I already started thinking about a possible idea for my project and considered creating something with the ultrasonic distance sensor later on.
For me, it is sometimes still difficult to keep up with all the new information and components, especially since many of the concepts are completely new to me. However, it is interesting to see how all the parts work together, and I am slowly starting to understand the basic ideas behind the circuits and the code.

# 11. November
In this session, we focused on movement and worked with servo motors. Using the Servo library and the myservo.write function, we controlled the angle of a servo motor connected to a small “wing.” Later, we used a potentiometer to make the movement smoother and adjust the speed. We also experimented with a joystick to control two servos at the same time by mapping the X and Y values. This helped me better understand how sensors can be used to control motion and create more interactive systems.
